## 概要

ゲーム配信プラットフォームである[Steam](https://store.steampowered.com/)でプレイしたゲームを一覧表示することのできるウィジットを作成し、その HTML 埋め込みコードを取得できます。

## 詳細

対象プレイヤーの ID を入力することでそのプレイヤーのプレイ時間の長いゲームを降順に表示します。  
ウィジットからはプレイヤーのプロフィールページ、各ゲームのストアページへアクセスできます。
